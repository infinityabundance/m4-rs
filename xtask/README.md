# xtask

Project maintenance tasks for m4-rs. Rust-native task runner (no Python).

Run `cargo xtask` for available commands: check, fmt, clippy, test, oracle, compare, gnu-compare, generate, receipts, claims, cleanroom, fuzz, smoke, bench, status.
